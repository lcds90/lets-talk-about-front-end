# CSS Grid 🎨

::: info 🚀 O que é?
O CSS clamp() é uma função útil para criar valores responsivos que se adaptam ao tamanho da viewport.
:::

::: tip 🤔 Para que serve?
Com o clamp(), você pode definir um valor mínimo, um valor ideal (preferencial), e um valor máximo. Isso é especialmente útil para criar larguras (width) que se ajustam de forma fluida entre diferentes tamanhos de tela.
:::
